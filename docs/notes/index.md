# 心得笔记

记录使用 Claude Code 进行 Verilog 开发过程中的经验、踩坑和思考。

## 笔记列表

> 这里会持续更新使用心得。每次有新的经验只需在 `docs/notes/` 下新建 `.md` 文件即可自动加入侧边栏。

---

### 如何添加笔记

1. 在 `docs/notes/` 目录下创建新的 `.md` 文件，如 `2026-04-27-fifo-debug.md`
2. 用 Markdown 写下你的心得
3. 提交并推送，网站自动更新

### 笔记格式建议

```markdown
# 标题：一句话总结学到的内容

**场景**：当时在做什么
**问题**：遇到了什么困难
**解决**：Claude Code 如何帮忙解决的
**心得**：学到了什么，下次该怎么做
```
